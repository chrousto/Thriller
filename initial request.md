1. A type in the form of an icon

3. The time
4. The body

6. A potential link to an attachement, often an image
7. A section with the reply I potentially made to the person when it's an answer to an interaction.

This list is split into several pages linked at the bottom of the page. More messages will flow in in the future

 3.1 Text in the body
 3.2 Date
 3.2 Type (the icon of point 1.)
 3.3 Sender
4. I want to be able to display the linked image inline
